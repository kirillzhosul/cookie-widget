Cookie Widget

What is that?

Cookie widget - is a simple widget that notifies users about site cookies usage. Widget does not includes any tracking or server-side, being deployed only using GitHub pages! (Static only)
Read more in repository README here!

How widget is looks like?

It is actually connected and used here!

How to use this widget?

Add this line inside your head HTML tag:

If you want use minified version, use widget.min.js,
for configurable widget (with required own request to show) use widget.c.js, see below how to configure.
